# Minimal_BSP_-Bare-Metal-_with_UART
This project demonstrates a minimal Board Support Package (BSP) for an ARM Cortex-M3 target. It includes- Startup code (Reset Handler, vector table), Linker script (memory layout), UART driver (polling-based), Bare-metal application (main.c), Execution using QEMU (lm3s6965evb board), The system prints boot messages and supports UART RX debug mode.
